# Resistor

![](data/screenshot_tui.png)

A program for calculating four-band resistors

## Building

- **Dependencies:** `rustc`, `cargo`, `ncurses`

### Build

```bash
cargo build --release
```

### Run

```bash
./target/release/resistor
```
